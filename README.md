#  java-oracle-lucene-search-demo
A demo for searching using OracleDB and Apache Lucene

## Installations
### OracleDB
Install OracleDB at http://www.oracle.com/technetwork/database/database-technologies/express-edition/downloads/index.html

Create user for OracleDB and configure in OracleCon.java
